# Formix

Formix, kullanıcıların günlük fitness ve sağlık süreçlerini takip edebilmesi için geliştirilmiş web tabanlı bir spor ve beslenme platformudur.

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- BMI hesaplama sistemi
- Kalori hesaplama
- Diyet planı oluşturma
- Egzersiz takip sistemi
- Hedef belirleme
- İlerleme takibi

## Kullanılan Teknolojiler

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Proje Amacı

Kullanıcıların spor, beslenme ve sağlık süreçlerini daha düzenli şekilde yönetebilmesini sağlamak.

## Geliştirme Ortamı

- WampServer
- phpMyAdmin
- Sublime Text
